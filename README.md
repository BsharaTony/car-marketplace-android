# CarMarket (Final Project - Mobile Apps Development)

## Overview
A simple car marketplace prototype demonstrating Android fundamentals:
- Dynamic car list using RecyclerView
- Search and sort by price
- Favorite toggle per item
- Car details page with back navigation
- Contact seller opens the phone dialer

## Tech Stack
- Java
- Android Studio
- RecyclerView + Adapter/ViewHolder
- XML layouts

## Pages Implemented
1. Main page: car list (RecyclerView) + search + sort + favorites
2. Details page: car title/price + back button + contact seller (dialer)

## Notes
- No data persistence (in-memory list only)
- No online/API capabilities

## How to Run
Open the project in Android Studio and press Run on an emulator/device.
